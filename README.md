# Tratamento de Exceções em Java ☕

## 📌 Sobre o Projeto
Este é um projeto prático focado na compreensão e aplicação do tratamento de exceções (`Exceptions`) na linguagem Java. O objetivo principal é garantir que a aplicação saiba lidar com erros e fluxos inesperados de forma limpa e segura.

Além do foco no código, este repositório também está servindo como um laboratório prático para testes e consolidação dos meus conhecimentos em **versionamento de código** utilizando Git e GitHub.

## 🛠️ Tecnologias Utilizadas
* **Java:** Lógica principal e tratamento de erros (try-catch, throw, exceções personalizadas).
* **IDE:** Eclipse
* **Versionamento:** Git e GitHub

## 🎯 Objetivos de Aprendizado
* Entender a hierarquia de exceções no ecossistema Java.
* Praticar a separação entre a lógica de negócios e a lógica de tratamento de falhas.
* Dominar os comandos básicos e o fluxo de trabalho do Git (commits, push, pull, branches).
